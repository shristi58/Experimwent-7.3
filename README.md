# Real-Time Chat Application Using WebSocket (Socket.io)

## Objective
The goal of this project is to build a real-time chat application using **Node.js**, **Express**, **Socket.io**, and **React**.  
This helps in understanding **two-way communication** between the server and clients, managing connected users, and updating the UI instantly as messages are exchanged.

---

## Features
- Real-time communication using **WebSocket (Socket.io)**  
- Multiple users can chat in the same room simultaneously  
- Displays messages instantly without page refresh  
- Shows user name and message timestamp  
- Works across multiple browser tabs/windows
